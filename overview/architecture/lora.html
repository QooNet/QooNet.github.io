LoRa Wireless Communication

For synchronous timing applications, the QooNet timing system uses

  • a single timeline transmitter
  • broadcasting to multiple timing Receivers

We extended our wireless philosophy to the timing system by using 436 MHz LoRa (Long Range) radio as the timing system carrier.

  • LoRa is a proprietary spread spectrum modulation scheme.
  • The transceiver we used is the Adafruit Feather M0 RFM95 LoRa Radio which costs about 350 SEK per unit.
  • We tested the range of the radio to over 150 meters without a directional antenna.
  • Synchronization to within 10 microseconds can be achieved
