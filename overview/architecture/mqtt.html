MQTT

A very popular IoT communication protocol is MQTT

  • Message Queue Telemetry Transport
  • ISO standard (ISO/IEC PRF 20922).

MQTT was invented by IBM in 1999 for connecting oil pipelines over satellite connection and is now an open source platform.

  • It is designed to be reliable, data-agnostic, and lightweight.
  • It is quickly becoming the protocol of choice for IoT.

MQTT offers natural data pooling

  • With the subscribe feature of MQTT, the device controllers are essentially screened from redundant requests from many users.
  • Users who subscribe only get new data when there is a new message

MQTT provides and additonal layer of security

  • to publish a message to a topic, the user needs a "key" and "secret" to access the MQTT broker
  • which provides central point to implement role based access screening.
