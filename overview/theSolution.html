The Solution - YggdrasilView - an IoT Control System

YggdrasilView is an easily customizable control platform

  • specifically targeted at the rapidly changing needs of small, high-tech startups
  • Based on IoT (Internet of Things) technology

YggdrasilView is:

  • Reliable - small form factor computers placed close to control points
  • Non-invasive - wireless technology
  • Low cost - based on the IoT revolution
  • Easily modifiable - Node-Red graphical code environment
  • Mobile - User interface is web based
  • Open source for long term growth
  • Secure - Authentication based on bcrypt hashing
