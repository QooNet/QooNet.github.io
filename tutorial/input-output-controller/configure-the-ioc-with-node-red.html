Configure the IOC With Node-Red

  • Import the tutorial flow libraries into the BBBW

    ssh into the BBBW as root and change the directory to library directory for your node-red installation and clone the YggdrasilView flows repository 

    cd /root/.node-red/lib
git clone https://github.com/yggdrasilView/flows.git
  • Open the tutorial flow from the Libarary

    Open a browser and connect to Node-Red at <hostName>.local:1880 where <hostName> is the host name of your BBBW Click on the three horizontal bar icon on the upper right hand side of the node-red desktop to open the menu. Navigate to Import>Library>tutorial>ioc>greenLedFlow.

    The nodes for controlling the green LED appear. Place the nodes how you would like on the screen. Rename the Flow tab by double clicking on the flow tab and changing the name in the name field.

  • Edit the MQTT information

    The MQTT broker username and password needs to be entered into the MQTT nodes. Double click on any MQTT node. (they are in light purple) For example double click on the node labeled "green/intensity/get". A dialog panel will open on the right side of the screen. Next to the Server field, there is a pencil icon for editing the MQTT broker information. Edit the MQTT broker information with the data for the MQTT broker you setup in the previous section of the tutorial.

  • Deploy the Flow

    Once the MQTT information has been entered, click on the Deploy button in the upper right hand corner of the screen. All the blue dots should disappear from the screen and green connected boxes should appear underneath the MQTT flows and the gpio flows.

  • Test the flow

    Press the green LED button on the test device and observe messages generated on the MQTT broker.
