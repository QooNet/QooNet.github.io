Change the Passwords

  • With the backup SD card removed, connect the BBBW to your linux computer with the micro-USB card.
  • Log into the BBBW as root 
    ssh root@192.168.7.2

    In the initial configuration there is no password required for root

  • change the root password to your desired password by typing 
    passwd

    and follow prompts

  • Logout of root 
    exit
  • Log in as debian which is the only other initial user configured on the BBBW 
    ssh debian@192.168.7.2

    The initial password for debian is temppwd. Change the password as you did for root. You can have a different password for debian if you would like
