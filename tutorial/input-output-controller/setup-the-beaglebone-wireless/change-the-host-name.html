Change the host name

The default host name of the BBBW is beaglebone. It is best to change this to something different so it can be uniquely identified on the network.

  • Edit the host file using the nano text editor by typing 
    sudo nano /etc/hostname

    and change beaglebone to the desired host name. Cntrl-x will save the changes

  • Edit the hosts file using the nano text editor by typing 
    sudo nano /etc/hosts

    and change every instance of beaglebone to the desired host name. Cntrl-x will save the changes

  • Reset the BBBW by pressing S1 (or power cycle) and the new host name should appear on the prompt
