Test Device Design

We will build a simple device that can demonstrate local and remote control and remote monitoring. The device will consist of two LEDs, one red, one green, and two push buttons, again one red, one green. The requirements for the device are:

  • When one of the push buttons is pressed, the corresponding LED will toggle
  • The toggle state of an LED should be viewable on the web application
  • The toggle state of an LED can be changed by the web application
  • The intensity of an LED can be changed by the web application
  • The intensity of an LED will be displayed on the web application

The circuit that fulfills these requirements is shown below. The Fritzing circuit can be found here.

  • The intensity of the LEDS can be controlled be connecting the anode of the LEDs to Pulse Width Modulator (PWM) pins on the Beagle Bone Black Wireless.
  • From the pinout diagram of the Beagle Bone, there are 7 PWM pins available.
    • We will use P9_14 for the green LED and P9_16 for the red LED.
    • We will use P9_1 as the ground pin and P9_3 as the voltage pin
